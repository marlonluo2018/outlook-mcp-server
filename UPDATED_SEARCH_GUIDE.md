# Outlook MCP 搜索指南

## 搜索方式

### 1. 单词搜索（不带引号）

当您输入不带引号的搜索词时，系统会将其拆分为多个单词进行搜索。

#### AND 逻辑（match_all=true）
使用`match_all=true`参数，系统会查找包含**所有**单词且这些单词彼此接近的邮件。

**示例请求：**
```json
{
  "params": {
    "search_term": "red hat partner day",
    "match_all": true
  }
}
```

**特点：**
- 查找包含"red"、"hat"、"partner"和"day"所有单词的邮件
- 这些单词必须在邮件内容中彼此接近（默认50个字符内）
- 结果更精确，避免包含所有单词但内容不相关的邮件

#### OR 逻辑（match_all=false）
使用`match_all=false`参数，系统会查找包含**任意一个**单词的邮件。

**示例请求：**
```json
{
  "params": {
    "search_term": "red hat partner day",
    "match_all": false
  }
}
```

**特点：**
- 查找包含"red"、"hat"、"partner"或"day"中任意一个单词的邮件
- 结果更广泛，通常返回更多邮件

### 2. 精确短语搜索（带引号）

当您输入带引号的搜索词时，系统会查找包含这个完整短语的邮件。

**示例请求：**
```json
{
  "params": {
    "search_term": "\"red hat partner day\"",
    "match_all": true
  }
}
```

**特点：**
- 查找包含完整短语"red hat partner day"的邮件
- 结果最精确，只返回完全匹配的邮件

## 搜索结果对比

以搜索"red hat partner day"为例：

| 搜索方式 | 结果数量 | 精确度 | 适用场景 |
|---------|---------|--------|---------|
| 单词搜索（match_all=true） | 13 | 高 | 查找相关主题，但不需要完全匹配 |
| 单词搜索（match_all=false） | 278 | 低 | 广泛搜索，查找任何相关内容 |
| 精确短语搜索 | 10 | 最高 | 查找特定短语或专有名词 |

## 最佳实践

1. **查找特定主题**：使用`match_all=true`的单词搜索
2. **查找特定短语或专有名词**：使用带引号的精确短语搜索
3. **广泛搜索**：使用`match_all=false`的单词搜索

## 注意事项

- 系统现在会自动处理单词的接近度，避免返回包含所有单词但内容不相关的邮件
- 对于专有名词或特定短语，建议使用引号进行精确搜索
- `match_all`参数只对不带引号的搜索词有效
